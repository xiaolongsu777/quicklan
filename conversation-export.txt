Assistant:
现在点“更新”，会提示最新版，不会退回到 0.1.1。

原因是当前实现的版本比较会把 0.1.1-b 清洗成数字版本 0.1.1 来比较：
- GitHub 最新：0.1.1
- 本机当前：0.1.1-b
- 比较结果：相等
- 前端提示：当前已是最新版 0.1.1-b

它不会下载 GitHub 上的 0.1.1 安装包，也不会降级。

严格语义化版本里，0.1.1-b 低于正式版 0.1.1，但当前逻辑是有意偏保守：同一数字版本不自动覆盖或降级。
